# Faculty Personal Details Analysis

## Overview
The Personal Details page displays the professional profile of the faculty member, similar to the student profile but tailored for staff attributes.

## Key Features
-   **Profile Card**: High-impact visual card with Avatar, Name, Designation, and Employee ID.
-   **Info Grid**: Categorized contact and professional details.
    -   **Contact**: Email, Phone, Address.
    -   **Academic**: Department, Qualification, Specialization.
    -   **Experience**: Joining Date, Experience (Years).
-   **Update**: Button to "Update Details" (mock navigation).

## UI Components
-   **Profile Hero**: Large banner style card for the main profile.
-   **Details List**: Icons and text for attributes like Email, Phone, etc.

## Data Integration
-   **Faculty Data**: Fetched via `getFaculty`. Renders the profile of the currently logged-in user.
