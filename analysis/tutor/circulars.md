# Tutor Circulars Analysis

## Overview
A targeted view of announcements specifically relevant to Tutors (e.g., administrative deadlines, exam duty rosters).

## Key Features
-   **Targeted Content**: Filters circulars where audience is 'Tutors' or 'All'.
-   **Management**:
    -   Search capabilities.
    -   Download attachments.
    -   Priority indicators.

## UI Components
-   **Announcement List**: Card-based feed of circulars.
-   **Priority Badges**: Visual cues for urgency.
-   **Sync**: Button to "Sync with Students" (mock logic for reposting/sharing).

## Data Integration
-   **Circulars**: `getCirculars` filtered by audience.
